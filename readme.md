## 実行方法

1. `init_db.py` を実行してデータベースを初期化します。
2. 次に、以下のコマンドで Flask アプリケーションを起動します:

    ```bash
    flask run
    ```

3. ブラウザで `http://127.0.0.1:5000` にアクセスしてください。
